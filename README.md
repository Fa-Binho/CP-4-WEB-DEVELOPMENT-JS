# EcoTrend - E-commerce Sustentável

## Descrição

E-commerce especializado em produtos sustentáveis e ecológicos, desenvolvido para o Check-point 04 de Engenharia de Software - Web Development with JS.

## 🌱 Objetivo

Implementar funcionalidades interativas com JavaScript: manipulação do DOM, armazenamento local e trabalho com JSON.

## 🛍️ Produtos

- **Roupas Sustentáveis**: Vestuário orgânico e reciclado
- **Beleza Natural**: Produtos veganos e cruelty-free  
- **Casa Sustentável**: Itens ecológicos para o lar
- **Tecnologia Verde**: Dispositivos de baixo impacto ambiental

## 🚀 Funcionalidades

### DOM e Interface
- Carrinho de compras dinâmico (sidebar)
- Sistema de filtragem por categoria e preço
- Renderização dinâmica de produtos

### Armazenamento e Dados
- Persistência do carrinho com localStorage
- Carregamento de produtos via JSON
- Estado mantido entre sessões

### Experiência do Usuário
- Notificações visuais instantâneas
- Processamento assíncrono de compras
- Design responsivo e animações suaves

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna
- **JavaScript ES6+** - Lógica e interatividade
- **Font Awesome** - Ícones
- **Google Fonts** - Tipografia
- **LocalStorage** - Armazenamento local
